## Image Specifications

All images should:
- Be optimized for web (preferably under 200KB)
- Use PNG format for quality
- Have transparent backgrounds where appropriate
- Follow the app's color scheme (orange/prakash-primary tones)
- Be culturally appropriate and respectful
- Look good in both light and dark modes

## How to Replace Placeholders

1. Generate images using AI tools (Midjourney, DALL-E, Stable Diffusion)
2. Save with the exact filenames mentioned above
3. Place in `/public` folder
4. The Next.js Image component will automatically optimize them
